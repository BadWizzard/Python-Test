# Добро пожаловать в рабочие заметки

Это твой личный vault для рабочих заметок.

## Структура папок

- **daily/** - ежедневные заметки
- **projects/** - заметки по проектам
- **meetings/** - заметки со встреч
- **ideas/** - идеи и мысли
- **archive/** - архив старых заметок
- **templates/** - шаблоны для заметок

## Горячие клавиши

- `Cmd/Ctrl + Shift + D` - создать ежедневную заметку
- `Cmd/Ctrl + Shift + T` - вставить шаблон

## Шаблоны

Доступные шаблоны:
- **daily-note** - для ежедневных заметок
- **project** - для проектов
- **meeting** - для встреч
- **idea** - для идей

## Начало работы

1. Создай ежедневную заметку (`Cmd/Ctrl + Shift + D`)
2. Используй шаблоны для структурирования заметок
3. Связывай заметки через `[[ссылки]]`
4. Используй теги для категоризации
